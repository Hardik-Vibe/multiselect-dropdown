Primeng Multiselect

  • We need to extend base component to support tree view. It will get complex with level.
  • Manually we need to manage root selection.
  • It will have drop down look and feel.
  • It has support of virtual scroll.
  • 2 way binding (for child we can use checkbox with ngModel).

Primeng Tree

  • It provides n level tree structure with different selection modes.
  • Component will handle all selection propagations.
  • We need to use overlay component to create drop down.
  • It has support of virtual scroll.
  • 2 way binding.

Ngx Treeview

  • It provides n level tree structure.
  • Component will handle all selection propagations.
  • It will have drop down look and feel but it is dependent on Bootstrap 4.
  • It has support of virtual scroll.
  • We need to update item object and need to call correctChecked method of TreeviewItem class
  • It has strict type for item array.
